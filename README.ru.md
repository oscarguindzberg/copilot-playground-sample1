# Менеджер задач (Доступный и адаптивный)

Это простой и доступный веб‑приложение для управления задачами со следующими возможностями (теперь используется светлая цветовая палитра):

- Добавление задач
- Отметка задач как выполненных
- Удаление задач
- Сохранение в LocalStorage
- Горячие клавиши + доступные живые обновления
- Адаптивный макет и современное оформление
 - Фильтрация: показать Все / Активные / Завершённые задачи
 - Тема: Светлая (по умолчанию). Интерфейс использует светлую палитру для лучшей читаемости при ярком освещении.

## Файлы

- `index.html` — семантическая разметка и структура
- `styles.css` — современные адаптивные CSS стили
- `script.js` — логика приложения, сохранение и функции доступности

## Запуск локально

Откройте `index.html` в браузере. Сервер не требуется. Для разработки можно запустить простой статический сервер, например на Python:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000 в браузере
```

## Доступность и клавиатурные сокращения

- Включены корректные подписи и состояния фокуса.
- Область `aria-live` сообщает об изменениях (задача добавлена, выполнена, удалена).
- Нажмите `N` или `/`, чтобы быстро сфокусировать поле добавления задачи.
- Используйте `Tab` и `Shift+Tab` для перехода между интерактивными элементами.
- Нажмите `Enter` или `Space` на метке задачи, чтобы переключить её состояние (можно также использовать чекбокс).
 - Нажмите `N` или `/`, чтобы быстро сфокусировать поле добавления задачи.
 - Используйте `Tab` и `Shift+Tab` для навигации. Используйте Enter/Space для активации кнопок.
 - Кнопки фильтра (Все / Активные / Завершённые) доступны для фокуса и активации с клавиатуры.

## Примечания

- Задачи сохраняются в `localStorage` под ключом `task-manager:tasks:v1`.
 - Задачи сохраняются в `localStorage` под ключом `task-manager:tasks:v1`.
 - Выбранный фильтр сохраняется под ключом `task-manager:filter:v1`.
- Простота задумана изначально — можете расширить: редактирование, сортировка, дополнительные фильтры, импорт/экспорт.
